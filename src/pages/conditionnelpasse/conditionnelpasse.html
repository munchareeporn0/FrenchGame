Conditionnel passé
Le conditionnel passé ใช้อย่างไร ...
1. ใช้ในโครงสร้าง Si hypothétique แบบที่ 3 (Si + plus-que-parfait, conditionnel passé) เพื่อพูดถึงเหตุการณ์ในอดีตที่เราไม่สามารถกลับไปแก้ไขเปลี่ยนแปลงได้
• Si Marie était partie plus tôt, elle serait arrivée à l'heure. ความหมาย : ถ้า Marie ออกไปเร็วกว่านี้ ก็จะถึงตรงเวลา (แต่ในความเป็นจริง Marie ออกไปสาย และไปถึงที่หมายสาย)
ประโยคที่ใช้ si hypothétique สามารถแฝงคำตำหนิได้ด้วย เช่น
• Si tu n'avais pas passé autant de temps à regarder la télévision, tu aurais pu finir tes devoirs. ความหมาย : ถ้าเธอไม่ได้ใช้เวลาในการดูทีวีนานขนาดนี้ เธอก็คงทำการบ้านเสร็จไปแล้ว
2. ใช้เพื่อบอกความเสียใจเกี่ยวกับเหตุการณ์ที่ผ่านมาแล้ว (exprimer le regret)
มักจะใช้กับ V.devoir, V. ที่บอกความชอบ / ความต้องการ เช่น V.aimer, V.préférer, V.vouloir
ประธานที่ใช้มักเป็น Je และ Nous (เสียใจในสิ่งที่ตัวฉันหรือพวกเราทำ/ไม่ได้ทำ)
• Nous aurions préféré faire notre stage dans une pharmacie plutôt que dans une épicerie. ความหมาย : พวกเราอยากฝึกงานในร้านขายยามากกว่าในร้านขายของชำ (แต่ในความเป็นจริงพวกเราได้ฝึกงานในร้านขายของชำ)
3. ใช้เพื่อให้คำแนะนำที่แฝงไปด้วยคำตำหนิ (conseil + reproche)
มักจะใช้กับ V.pouvoir, V.devoir
• Il aurait pu faire l'effort de venir, c'était la moindre des choses. ความหมาย : เขาน่าจะพยายามมาสักหน่อยนะ มันเป็นสิ่งเล็กๆน้อยๆที่เขาควรจะทำ
• Tu n’aurais pas dû être si méchant. ความหมาย : เธอไม่ควรจะใจร้ายแบบนั้นนะ
4. ใช้เพื่อพูดถึงเหตุการณ์ในอดีตที่ไม่แน่นอนเป็นการคาดเดา Information incertaine
• Un avion se serait écrasé ce matin dans les environs de Dijon. ความหมาย : เครื่องบินน่าจะตกเมื่อเช้าแถวๆเมือง Dijon
